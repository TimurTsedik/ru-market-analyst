# ru-market-shared — общий справочный пак

Не инвокается на анализ. Общие справочники для остальных ru-market-скиллов:

- `references/instruments-rf.md` — классы инструментов РФ, оценка и риск-метрики
  (грязная цена/НКД, амортизация секьюритизаций, восстановление $-номинала,
  yield-to-worst), ИОС/структурки, рейтинговая шкала, G-спред.
- `references/sources.md` — иерархия источников, провенанс (источник + as-of), свежесть,
  sanity-gate, блок «Источники и аудит».
- `references/taxes-accounts.md` — ЛДВ, ИИС-3, купонный налог, валютная переоценка,
  налоговые лоты.

Другие скиллы ссылаются сюда как `../ru-market-shared/references/<file>.md`.

**Когда НЕ использовать:** на прямой запрос пользователя. Это внутренний пак — его
подгружают broker-parse / cbonds-research / market-data / risk-strategy по необходимости.
