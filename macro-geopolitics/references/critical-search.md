# Критический поиск, свежесть и кросс-верификация (official ≠ реальность)

Урок-основа: **один снипет поиска — это зацепка, не факт.** Material-число (то, что двигает
вывод/решение) проходит четыре гейта: **свежесть = сегодня**, **≥2 независимых источника или
явный single-source-флаг**, **первоисточник предпочтительнее**, и **проверка расхождения
официального с физикой**. При блоке источника — **не выдумывай**. Этот файл — обязательный
для фаз 1 (режим) и 3 (индикаторы).

## 1. Свежесть-гейт (freshness = сегодня)
- Для **ежедневных рядов** (курс ЦБ, RGBI, недельная инфляция, цены MOEX) бери значение **на
  текущую дату сессии**, а не первое из выдачи. Снипет, датированный −N дней, — **устаревший**:
  перетяни «на сегодня» (`курс ЦБ на <ДД.ММ.ГГГГ>`), не выдавай вчерашнее за актуальное.
- Не путай **дату публикации статьи** с **датой данных**. Каждое material-число несёт
  `as-of = дата данных + дата чтения`. Расходятся — пометь.
- Быстро меняющееся (курс/спред/новостной фон/режимные акты) устаревает за дни; если между
  запросом и ответом прошло время — переспроси свежесть, не предполагай.

## 2. Мульти-источник / запрет одиночного снипета
- Material-число — **≥2 независимые линзы** ИЛИ явный флаг `single-source / не подтверждено`.
- **Первоисточник > агрегатор > пересказ.** Для ставки/курса/инфляции — cbr.ru / Росстат /
  MOEX напрямую (рецепты — `../market-data/`), а не чужой пересказ снипета.
- Если две линзы расходятся в разы — это не «усреднить», а **разобраться почему** (разные
  даты? разные определения? одна устарела?).

## 3. Запрет фабрикации при блоке источника
- Источник отдал **403 / paywall / пустой JS** → **НЕ подставляй число по памяти/«примерно».**
  Порядок: (а) альтернативный источник того же показателя; (б) не вышло → пометь
  **«не верифицировано вживую»** и **попроси пользователя прочитать с открытой у него
  страницы**. Дыра с пометкой честнее правдоподобной цифры из воздуха.
- Это прямое следствие no-hardcode: лучше явно «не достал», чем тихо выдуманное.

## 4. Official ≠ effective ≠ street (ядро для военной/контролируемой экономики)
В стрессовой/контролируемой экономике официальные агрегаты систематически расходятся с тем,
по чему живёшь. **Всегда тяни обе величины и называй разрыв** — он сам по себе вывод.

| Официальное (учётное) | Эффективное / street (по чему реально живёшь) | Где взять реальное |
|---|---|---|
| Курс ЦБ (фиксинг) | **Наличный курс обменника** (купля/продажа + спред) | cash.rbc.ru / банки; при блоке — попросить у пользователя |
| Headline CPI (Росстат) | **Физика дефицита**: талоны, нормирование, очереди, пустые полки, бензин по карточкам | новостной фон + прямой опыт пользователя |
| Управляемый FX-режим | Реальная **доступность конвертации/вывода** (капконтроль) | официальные акты + практика |
| «Рынок жилья растёт» (индекс цен) | **Объём сделок + срок экспозиции + дисконт при торге** (ликвидность уходит раньше цены) | Росреестр / объявления |
| Ставки по вкладам (витрина) | Реальная доходность net инфляции, по которой реально размещаешь | банки + физ. инфляция |

**Правило приоритета при расхождении:** в стрессе/контроле **верь физике, не индексу.**
Талоны на бензине бьют «официальную инфляцию 5%»; наличный курс бьёт фиксинг; срок продажи
квартиры бьёт индекс цен. Когда витрина спокойна, а физика кричит — вес на физику, и явно
скажи об этом.

## 5. Anti-anchoring — симметрично, в обе стороны
- **Не дисконтируй прожитый опыт пользователя** как «катастрофизацию» без проверки фактом
  (в этой сессии топливный кризис/талоны подтвердились — нарратив был прав).
- **Не принимай официальные числа** как истину (та же сессия: headline-инфляция занижает
  физику дефицитов).
- Обновляйся по уликам **в обе стороны** и **явно говори, что изменил** («снимаю свою
  формулировку», «это поправка к моему тезису»). Калибровка важнее правоты.

## 6. Чек-лист перед любым material-выводом
- [ ] Число **на сегодня** (не из устаревшего снипета); as-of проставлен.
- [ ] **≥2 независимых** источника ИЛИ флаг single-source.
- [ ] **Первоисточник** предпочтён пересказу.
- [ ] Источник заблокирован → **не выдумано**, помечено/спрошено у пользователя.
- [ ] Снята пара **official ↔ street**; расхождение названо; при стрессе вес на физику.
- [ ] Обновление по уликам — **в обе стороны**; сказано, что изменилось.

*Один снипет — зацепка, не факт. Свежесть на сегодня; первоисточник; не выдумывай при
блоке; верь физике, а не витрине; калибруйся в обе стороны.*
