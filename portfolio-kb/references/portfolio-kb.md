# База знаний о портфеле — схема и правила

Персистентный артефакт по запросу. **Приватность:** хранить **вне git-репозитория**
(напр. `~/portfolio-kb/`), номера счетов/договоров/ФИО **маскировать** (до последних цифр).

## Структура файла
- **Мета:** версия схемы; as-of по источникам (cbonds/MOEX/ЦБ/брокеры); список счетов
  (замаскированы, с типом ИИС/брокерский).
- **Позиции:** брокер · счёт · ISIN · наименование · кол-во · оценка ₽ · класс/корзина ·
  валютный тег · источник+as-of · флаги (амортизация/USD/структурка/плечо/отозван рейтинг).
- **Инструментальные данные (cbonds):** по ISIN — цена%/НКД, рейтинг (низший)+статус,
  оферта/амортизация/непогашенный номинал, YTW/дюрация, ликвидность.
- **Агрегаты:** взвеш. дюрация, доходность net-of-tax, низший рейтинг, доли корзин,
  валютная нога, плечо, концентрация (эмитент/сектор/топ-5).
- **Источники и аудит:** по ключевым цифрам — источник → as-of → уверенность.

## Дифф между прогонами
При повторном прогоне: читать прошлый файл → мёржить новые данные → блок диффа:
появилось / ушло / изменилось (кол-во, оценка, доля, рейтинг, флаги). Помечать
mixed-dates, если источники на разные даты.

## Версионирование
Поле версии схемы в начале файла; single-user — без протокола concurrency
(перечитать и перезаписать целиком).
