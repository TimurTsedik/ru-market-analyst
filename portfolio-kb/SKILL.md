# portfolio-kb — база знаний о портфеле (опционально)

Персистентный снимок портфеля + дифф между прогонами. Схема и правила —
`references/portfolio-kb.md`.

**Только с согласия пользователя** (данные счетов чувствительны). Хранить **вне
git-репозитория**, номера счетов/договоров **маскировать**.

**Выход (handoff):** канонический файл состояния (позиции + cbonds-данные + флаги +
as-of + источники + версия схемы) и блок диффа (что появилось/ушло/изменилось).

**Когда НЕ использовать:** обычный разовый разбор без запроса на сохранение/сравнение.

Общие справочники: `../ru-market-shared/references/`. Если их нет — попроси установить
все ru-market-скиллы.

*Это разбор, не ИИР. No-hardcode.*
