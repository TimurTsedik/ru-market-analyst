# Рыночные данные: MOEX ISS + ЦБ (рецепты)

Цены/ликвидность/кривая и макро. Иерархия источников — в `../../ru-market-shared/references/sources.md`; здесь — как
именно тянуть. Через `web_fetch` (без авторизации, чище браузера). Каждое число —
с as-of; рынок закрыт/устарело → пометь `*`.

## MOEX ISS — фонды, эквити, исполнимая цена

- **Борды:** фонды (БПИФ/ETF) обычно **TQTF**, эквити — **TQBR**. Борд **не
  универсален** — если неизвестен, смотри boards-метаданные ISS и бери основной.
- **Сначала определи engine/market/board** бумаги через метаданные
  (`https://iss.moex.com/iss/securities/{SECID или ISIN}.json`) — НЕ гони все фонды
  через `markets/shares`.
- **Цена:**
  `https://iss.moex.com/iss/engines/{engine}/markets/{market}/securities/{SECID}.json?iss.meta=off&iss.only=securities,marketdata`
  → бери `LAST`; при отсутствии/устаревании — `MARKETPRICE` / `LEGALCLOSEPRICE` /
  `WAPRICE` / `LCLOSEPRICE` / `PREVADMITTEDQUOTE` (зависит от борда). Фиксируй
  `BOARDID` + дату.
- **SECID по ISIN** (если тикер неизвестен):
  `https://iss.moex.com/iss/securities/{ISIN}.json` → secid + рынки/борды.
- **Фонды денежного рынка** (LQDT/TMON и т.п.) — цена пая на TQTF; стоимость позиции
  = кол-во паёв × цена.

## MOEX — кривая ОФЗ (для режима)

- Сними ликвидные **ОФЗ-ПД (фикс-купон)** ближайшие к ~2 / ~5 / ~10 годам — их
  YTM/дюрацию (cbonds или ISS). **Исключи флоутеры (ОФЗ-ПК) / линкеры (ОФЗ-ИН) /
  амортизируемые** — иначе смешаешь типы. Записывай фактическую дату погашения/дюрацию.
  Форма кривой = спред 2/10.
- **RGBI — контекст** (индекс гособлигаций MOEX), не сама кривая.

## ЦБ РФ

- **Ключевая ставка + сигнал:** cbr.ru (`hd_base/KeyRate`) — значение + дата
  вступления в силу + ближайшее заседание/среднесрочный прогноз.
- **Курсы:** официально — `https://www.cbr.ru/scripts/XML_daily.asp` (XML ЦБ);
  удобное JSON-зеркало `https://www.cbr-xml-daily.ru/daily_json.js` — как **fallback**.
  USD/CNY/EUR + дата (для оценки валютных/бивалютных бумаг).

## Маршрут по классам (кратко; полная иерархия — `../../ru-market-shared/references/sources.md`)

- Фонды / эквити / исполнимая цена / оборот → **MOEX ISS**.
- Термины / рейтинги / оферты / амортизация / проспекты → **cbonds**.
- Ставка / курс / инфляция / RGBI → **ЦБ / MOEX**.
- Стоимость портфеля (mark) → **рыночная оценка брокера** (первична).
