# market-data — живые рыночные данные РФ

Точечная выдача котировок/ставки/курса/кривой. Рецепты эндпоинтов —
`references/market-data.md` (MOEX ISS: фонды/эквити по борду + boards-метаданные, кривая
по ликвидным ОФЗ-ПД; ЦБ: ключевая ставка + курс через `web_fetch`). Иерархия источников —
`../ru-market-shared/references/sources.md`.

**Выход (handoff):** live-марки фондов/эквити, кривая (2/5/10-летние ОФЗ + спред 2/10),
ключевая ставка, курс — каждое с **as-of**; рынок закрыт/устарело → пометка `*`.

**Когда НЕ использовать:** разбор инструмента/рейтинг/проспект (→ cbonds-research),
парсинг отчётов (→ broker-parse), корзины/стратегия (→ risk-strategy).

Общие справочники: `../ru-market-shared/references/`. Нет — попроси установить все ru-market-скиллы.

*Это разбор, не ИИР. No-hardcode: цифры — вживую, не из памяти.*
