# cbonds-research — точечный ресёрч на cbonds

Разбор одной бумаги/эмитента по ISIN. Полная процедура — `references/cbonds-workflow.md`
(подключение к браузеру, поиск по ISIN со скриншотом выпадашки, **ISIN-gate**, чек-лист
полей со скриншота, скачивание проспектов через прямой `navigate` на URL → unzip → pypdf,
детект USD/флоутер/секьюритизация/структурка). Классы/метрики/рейтинги — в
`../ru-market-shared/references/instruments-rf.md`.

**Структурные ноты:** если на cbonds нет полного payoff — запроси у пользователя скрины
ноты из приложения (можно несколько) / term sheet / KID; если payoff неизвестен — класс
«структура/неликвид, payoff unknown», без точного YTW.

**Выход (handoff):** по ISIN — тип/класс, цена % + НКД, рейтинг (низший) + флаги
(Withdrawn), оферта/амортизация/непогашенный номинал, YTW/дюрация, ликвидность,
источник + as-of.

**Когда НЕ использовать:** парсинг сырых отчётов (→ broker-parse), живые котировки
фондов/эквити/курс (→ market-data), корзины/стратегия (→ risk-strategy), полный портфель
(→ ru-market-analyst).

Общие справочники: `../ru-market-shared/references/`. Нет — попроси установить все ru-market-скиллы.

*Это разбор, не ИИР. No-hardcode: данные — вживую с cbonds, не из памяти.*
