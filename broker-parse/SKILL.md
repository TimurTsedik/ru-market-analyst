# broker-parse — парсинг брокерских отчётов РФ

Точечный скилл: отчёты → чистые позиции. Полная процедура — `references/broker-reports.md`
(какой формат просить: **Excel-first**, но PDF для сделок/кэша/купонов/налога/счёта/плеча;
тулчейн-фолбэк python-venv `pypdf`/`xlrd`; сверка с итогом; дедуп; детект плеча).

**Выход (handoff):** по каждому счёту — позиции (брокер · счёт · ISIN · кол-во ·
оценка-если-есть) + кэш + признаки плеча + сверка с напечатанным итогом + пометка
mixed-dates. Номера счетов/договоров **маскируй**.

**Когда НЕ использовать:** оценка инструментов по cbonds (→ cbonds-research), живые цены
(→ market-data), корзины/стратегия (→ risk-strategy), полный сквозной разбор портфеля
(→ ru-market-analyst).

Общие справочники: `../ru-market-shared/references/`. Если их нет — попроси установить
все ru-market-скиллы.

*Это разбор, не ИИР. No-hardcode: рыночные данные бери вживую, не из памяти.*
