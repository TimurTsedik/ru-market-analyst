# RU Market Analyst — стратегический слой (design / spec) — ред. 2

Дата: 2026-06-21
Статус: ред. 2 после ревью codex (thread 019eeaf8), на подтверждении

## Цель

Усилить аналитический блок: добавить **стратегический движок** — итеративный цикл
«траинга» (детект режима → генерация вариантов → «а что если» → скоринг → итерация →
синтез) в связке с глубоким принципиальным анализом. Покрытие: облигации по режимам;
**полноценный** блок по акциям (с секторными линзами); ось «отчётность значит / не
значит». Исполнение — **гибрид** (цикл в уме по умолчанию, эскалация в мультиагентный
Workflow для крупных/спорных решений).

Подача: не хеджировать «не ИИР» в каждом абзаце. После каждого блока рассуждения —
**«Основание: …»** (данные / cbonds as-of / макро-источник / явное допущение). В
конце разбора — дисклеймер. Скоринг — **ординальный** (++/+/0/−/−−) + грубые
диапазоны со звёздочкой, без псевдо-оптимизации (входные данные indicative/устаревают).

## strategy-engine.md — протокол

### 0. Интейк-констрейнты (до генерации)
Собери/уточни: горизонт, потребность в ликвидности, риск-аппетит, тип счёта,
налоговые лоты/ЛДВ, лок-ап ИИС-3, желаемый RUB/FX-сплит, лимиты на эмитента/сектор,
базовый сценарий пользователя. Без этого движок сгенерит привлекательные, но
неприменимые стратегии. Источник целей подушки/тезиса — интейк-шаг скилла.

### 1–7. Цикл (траинг)
1. **Детект режима** (оси ниже) + уверенность + основание. Сигналы наследуют
   `sources.md`: каждый несёт as-of, флаг устаревания, уверенность.
2. **Мост от диагностики:** старт — от текущих корзин риска (S9). Кандидаты =
   движения, улучшающие концентрацию/дюрацию/FX-ногу/ликвидность/налог/ожид.
   доходность. Не переделывай диагностику — используй её.
3. **Генерация** 2–4 кандидатов; каждый проходит **eligibility-гейт**: налоговая
   обёртка, ЛДВ/FIFO, лок-ап ИИС-3, валютная переоценка, лимиты из интейка.
4. **«А что если»** — стресс каждого по библиотеке сценариев + флип режима.
5. **Скоринг (ординальный)** — по каждому сценарию ++/+/0/−/−− + грубый диапазон*;
   худший случай/дродаун, net-of-tax, **против бенчмарка** (текущая аллокация +
   денежный рынок/депозит-прокси + короткий ОФЗ/флоутер + базовый сценарий
   пользователя). Робастное предпочесть высоковариативному при отсутствии явного
   риск-аппетита.
6. **Итерация** — взять лучшего, закрыть слабый сценарий комбинацией/хеджем,
   перетест. **Стоп-рул (детерминированный):** максимум 2 итерации ИЛИ новый
   кандидат не даёт **материального** улучшения. Материальное = ≥1 ступень
   ординального скоринга хотя бы в одном сценарии ИЛИ улучшение худшего дродауна /
   снижение концентрации / налоговый эффект сверх порога (по умолчанию ~1% портфеля
   или явный порог пользователя).
7. **Синтез** — стратегия + сценарии (пережитые/проваленные) + основание каждого
   тезиса + дисклеймер. Глубокий принципиальный анализ (почему, от первых принципов
   + данные) вплетён по ходу.

### Оси режима + сигналы детекции
- **Цикл ставки** (рост/у пика/снижение/низко-стабильно): ключ + сигнал/прогноз ЦБ,
  RUONIA, инфляция/ожидания, форма кривой.
- **Форма кривой** (инверсия/плоская/нормальная/крутая): **ликвидные точки ОФЗ /
  кривая MOEX (zero-coupon/constant-maturity)**; RGBI — поддерживающий контекст, не
  сама кривая; спред 2/10.
- **Кредитный цикл** (спреды узкие/расширяются/широкие): G-спреды по рейтинговым
  корзинам, поток размещений, дефолты/реструктуризации, рейтинговые действия.
- **Инфляция** (рост/стабильна/дезинфляция): ИПЦ, инфл.ожидания, breakeven ОФЗ-ИН.
- **FX/санкции**: USD/CNY/RUB, премии/базис замещаек, CNY-ставки, платёжные цепочки.
- **Ликвидность** (норма/стресс; вынужденные потоки): объёмы, bid-ask, ставки
  денежного рынка/РЕПО.
- **Мета «отчётность значит?»** (доспецифицировано): кросс-секционная корреляция/
  широта, post-earnings abnormal return, ревизии прогнозов, дисперсия оценок и
  секторов, признаки вынужденных потоков (индексные ребалансы), доминирование
  санкционных/госсобытий. Низкая дисперсия + макро/новости рулят + вынужденные потоки
  → фундаментал не работает (торгуй режим/поток). Высокая дисперсия + реакция на
  отчёты + стабильное макро → работает (оценка/качество/отбор). Отдельно: качество
  отчётности (МСФО/РСБУ), разовые статьи, валютная переоценка, FCF/дивидендная
  способность ≠ headline-прибыль.

### Библиотека сценариев «а что если»
Ставка ±200 бп; твист кривой (steepen/flatten); расширение кредитных спредов **по
рейтинговым корзинам** (не только дефолт слабейшего); **real-yield шок ОФЗ-ИН +
инфляционный лаг**; дивидендный срез/приостановка выплат; просадка акций −20/−30% +
секторный шок; **сырьевой шок** для экспортёров; **санкции/обрыв платёжной цепочки**
для замещаек/евробондов; FX ±10–15% по чистой валютной ноге; **ликвидный хэйркат /
невозможность выйти по mark**; смена режима (фундаментал вкл/выкл).

### Эскалация в Workflow (гибрид, hard-gate)
По умолчанию — цикл в уме. Эскалируй, когда: крупный/сложный портфель, высокая ставка
решения, просьба «глубоко», близкие кандидаты. **Hard-gate:** мультиагентные линзы
запускаются ТОЛЬКО после заморозки диагностики; передай общий пакет фактов (единый
снимок портфеля, блок «Источники и аудит», налоговые констрейнты, набор сценариев) —
иначе агенты навыдумывают разные факты. Форма: параллельная генерация (стратег-линзы:
rate-first/credit-first/FX-first/защитная) → состязательный «а что если» (скептики
ломают каждый) → скоринг → синтез.

### Основание и дисклеймер
После каждого блока — `Основание: <данные/cbonds as-of/макро-источник/допущение>`.
Один дисклеймер в конце. Заменяет пер-тезисное хеджирование.

## strategies-bonds.md — плейбуки (по режимам, с гвардами)

- **Ставка растёт:** короткая дюрация, флоутеры, денежный рынок; избегать длинного фикса.
- **У пика ставки:** частично удлинять дюрацию (зафиксировать доходность до снижений),
  barbell; пик — не точка → частичность/итерации.
- **Снижение ставки:** длинный фикс выигрывает телом; реинвест-риск коротких/флоутеров.
- **Кривая инвертирована (с гвардом):** короткий конец платит, НО длинная дюрация
  может быть привлекательна до снижений, если длинный конец уже закладывает
  правдоподобную траекторию; опасна, если переоценятся срочная премия/Минфин/инфляция.
  Сравнивай carry/rolldown/ожид. траекторию/срочную премию **по горизонту**, не по
  фикс-правилу.
- **Кривая крутая (с гвардом):** rolldown привлекателен ТОЛЬКО при неизменной/
  благоприятной кривой, достаточной ликвидности и без спред-шока; в ОФЗ навес/дефицит/
  аукционы Минфина могут перебить. Считай carry + rolldown vs неблагоприятный
  параллельный/твист-шок.
- **Кредит — узкие спреды:** качество, не гнаться за спредом. **Широкие/паника:**
  выборочно спред в крепких именах (G-спред с поправками), сверять дефолтный цикл.
- **Инфляция растёт (с гвардом):** ОФЗ-ИН несут real-yield дюрацию, лаг ИПЦ ~3 мес,
  риск breakeven/ликвидности — **не** генерик-хедж; флоутеры хеджируют RUONIA/политику,
  не ИПЦ напрямую, и лагают, если инфляция скакнёт до реакции ЦБ.
- **FX (с гвардом):** замещайки/юань — это валютная нога-**кандидат**, дальше тест на
  кредит эмитента / право / налог (переоценка) / ликвидность / CNY-базис и CNY-ставки;
  не линейное «слабый рубль → покупай валюту».
- **Ликвидный стресс:** денежный рынок/качество, ждать; вынужденные продажи качества
  с дисконтом — возможность.

## strategies-equity.md — секторные линзы + ось отчётности (полноценно)

### Держать ли акции при высокой ставке
Сравни ожидаемую доходность акций с карри облигаций net-of-tax. Корректно:
`дивдоходность + shareholder yield (buyback) + ожид. рост прибыли/FCF + переоценка`
ИЛИ `earnings/FCF-yield как планка vs карри`. **Не** `E/P + рост + дивиденд` (двойной
счёт: дивиденд платится из прибыли). При высоком ключе планка для акций выше.

### Линзы по имени
ао vs ап; free-float/ликвидность; governance; санкционный статус; дивполитика и её
устойчивость; buyback; FCF; чистый долг/леверидж; capex; валютная переоценка; оценка
(P/E, EV/EBITDA, дивдоходность) с поправкой на разовые статьи и качество отчётности.

### Секторные линзы
- **Банки:** достаточность капитала, NIM, стоимость риска, суборды/перпы в структуре.
- **Нефтегаз:** цены, демпфер/НДПИ/пошлины, FX-выручка, дисконты сорта, санкции/логистика.
- **Металлурги/добыча:** сырьевой цикл, экспортные пошлины, FX, capex.
- **Энерго/коммуналки:** тарифное регулирование, индексация, capex/RAB.
- **Ритейл/потребитель:** реальные доходы, чувствительность к ставке, конкуренция.
- **Девелоперы:** ипотека/льготы, ставка, долговая нагрузка, escrow.
- **Телеком/IT:** регулирование, capex, импортозамещение, рост vs зрелость.

### Госриск (чек-лист)
Госучастие может значить: принудительный payout ИЛИ приостановку выплат, спецналоги/
windfall, capex/социальные мандаты, тарифное регулирование, санкционные ограничения.
Не приравнивай «госкомпания» к «надёжный дивиденд».

### Экспортёры (гварды)
«Слабый рубль = плюс» — НЕ автоматически: проверь FX-долг, экспортные пошлины, ценовые
потолки, санкционную логистику, валюту расчётов, рублёвую базу затрат, импортный capex,
сырьевой цикл. Часть выгоды от FX съедается налогами/оборотным капиталом.

### Режимы отчётности
- **«Не работает»:** торгуй поток/событие/позиционирование (индексные ребалансы,
  навес/выкупы, санкционные бинары, розничный моментум); снизить идиосинкратику →
  ликвид/индекс или сидеть.
- **«Работает»:** оценка/дисперсия/качество/отбор имён.
